# 🤖 AI 模型分析與 ISO 19650 QC 審查助手使用教學

本工具為 Revit 深度整合的 AI 輔助品管 (QC) 系統，旨在協助 BIM 經理與工程師以「自然語言」快速稽核並完善符合 **ISO 19650** 規範的資訊模型，同時兼顧高幾何安全性，防止模型被 AI 意外改壞。

---

## 1. ⚙️ 第一步：API 連線設定

本工具採用業界標準的 **OpenAI Chat Completions 相容 API 規格**。您可以自由對接雲端大模型或本地離線模型：

1. **開啟工具**：點擊 Revit 頂部 Ribbon 欄 `Development tools` ➔ `AI模型分析`。
2. **展開設定**：點擊視窗右上角 **「⚙ API 設定」**。
3. **輸入參數**：
   * **API Key**：您的金鑰（若使用本地 Ollama 可隨意輸入任意字元）。
   * **API Endpoint**：
     * *OpenAI*：`https://api.openai.com/v1/chat/completions`
     * *本地 Ollama (離線)*：`http://localhost:11434/v1/chat/completions`
     * *自訂中轉/轉接點*：輸入對應的 `/v1/chat/completions` 網址。
   * **Model Name**：模型名稱（例如 `gpt-4o`、`llama3` 等）。
4. **驗證連線**：點擊 **「確認連線」**。連線成功後設定將**加密儲存**在本地 `%LOCALAPPDATA%\DevelopmentTools\App\ai_settings.json`，下次開啟自動載入。

---

## 2. ⬇️ 第二步：讀取 Revit 數據 Context (雙模式)

點擊左下角 **「⬇ 讀取 Revit 資料 (選取/全專案掃描)」** 按鈕，系統會根據您當前在 Revit 中的選取狀態，智慧採取不同模式：

### 🅰️ 模式一：手動選取模式 (局部精確審查)
* **操作方法**：先在 Revit 中以滑鼠框選一或多個元件（例如：3 面牆、2 個房間），然後點擊讀取按鈕。
* **Context 內容**：AI 將僅獲取這些被選取元件的幾何資訊、材質、備註、尺寸與鋪磚縫隙參數。
* **適用場景**：局部磁磚鋪設衝突排解、特定元件參數修改、單一房間裝修確認。

### 🅱️ 模式二：免選取全域掃描模式 (BIM 品質 QC)
* **操作方法**：**不選取任何元件**，直接點擊讀取按鈕。
* **Context 內容**：系統將自動執行全專案模型大掃描，並提供以下資料：
  1. **ISO 19650 分類填寫率**：統計所有牆體與地板是否設定了 `Assembly Code`（即 Uniclass 或 Uniformat 分類代碼）與 `OmniClass`，並列出缺失編碼的元件 ID 清單。
  2. **空間裝修缺失清單**：按樓層統計房間數，自動抓出該樓層未填寫 `Base Finish` (地板裝修) 或 `Wall Finish` (牆面裝修) 參數的房間名稱與房號。
* **適用場景**：專案交付前的模型資料完整度稽核、樓層進度追蹤。

---

## 3. 💬 第三步：對話稽核與優化建議

讀取完 Context 後，您可以直接在對話框中發問，以下是推薦的實戰提問：

* **全域 QC 提問範例**：
  > 🗣️ *"幫我檢查各樓層裝修做好了沒，有哪些房間漏掉了？"*
  > 🗣️ *"專案裡有沒有牆面或地板漏掉了 Uniclass 分類代碼 (Assembly Code)？幫我抓出來。"*
  
* **局部優化提問範例**：
  > 🗣️ *"選中的這幾面牆，有漏寫備註嗎？有的話幫我填寫為 'Approved_QC'"*
  > 🗣️ *"分析所選地板的磁磚縫隙參數是否合理，若太寬幫我改成 2mm。"*

---

## 4. ⚡ 第四步：雙重安全確認執行修改 (防改壞機制)

為了解決 AI 意外改壞模型的問題，本工具實作了嚴格的**防改壞防線**：

1. **指令不默默執行**：AI 提出修改建議後，會以卡片形式在對話氣泡中顯示 **「⚡ 立即在 Revit 中執行 AI 修改」** 按鈕。
2. **原生確認對話框 (TaskDialog)**：點擊按鈕後，會彈出 Revit 原生的二次確認對話框，展示詳細的變更對照表：
   * **變更元件與 ID**。
   * **目標參數名稱**（如 `Assembly Code`、`Comments`、`Base Finish`）。
   * **【目前數值 (舊值)】 ➔ 【AI 優化新值】** 的清晰比對。
   * **長度單位自動換算**：若是幾何尺寸或縫寬參數，系統會自動將 Revit 內部的英呎換算為 **毫米 (mm)** 顯示，防範尺度誤差。
3. **同意或取消**：
   * 點選 **「Yes (同意)」**：在 Revit 主執行緒中啟動 Transaction 寫入模型。
   * 點選 **「No (不同意)」**：自動 Rollback，模型毫無變更，安全無虞。
