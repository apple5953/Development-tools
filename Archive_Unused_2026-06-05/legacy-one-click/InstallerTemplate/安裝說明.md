# Development tools Revit 外掛一鍵安裝包

## 給工程師的安裝方式

1. 先完整解壓縮這個資料夾，不要直接在 zip 裡執行。
2. 關閉 Revit 2024。
3. 雙擊 `install.bat`。
4. 重新開啟 Revit 2024。
5. 如果 Revit 跳出外掛載入提示，選擇 `Always Load`。

## 安裝位置

預設安裝到目前 Windows 使用者：

`%AppData%\Autodesk\Revit\Addins\2024`

安裝器會複製外掛 DLL 與相依檔案到：

`%AppData%\Autodesk\Revit\Addins\2024\RoomTileSystem`

並產生：

`%AppData%\Autodesk\Revit\Addins\2024\RoomTileSystem.addin`

## 解除安裝

雙擊 `uninstall.bat`，它會移除上述資料夾與 `.addin` 檔案。

## 管理員或全機安裝

如果要安裝給同一台電腦的所有 Windows 使用者，請用系統管理員身分開啟 PowerShell，切到本資料夾後執行：

```powershell
.\install.ps1 -AllUsers
```

全機安裝位置會是：

`%ProgramData%\Autodesk\Revit\Addins\2024`

## 注意事項

- 此版本是給 Revit 2024 使用。
- 請保持 `RoomTileSystem` 資料夾、`install.ps1`、`install.bat` 在同一層。
- `platform_config.json` 是 Google 驗證與白名單設定，請不要漏掉。
- 如果 Windows 封鎖下載的 zip，請在解壓縮前對 zip 右鍵、內容、解除封鎖。
